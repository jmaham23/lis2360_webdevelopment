Application-Layer Attack HTTP Flood

HTTP Flood is a common type of Application Layer Attacks. This method compels the server to allocate resources to serve the HTTP request. There are two types of requests used for HTTP flooding: GET Flood and POST Flood. The GET Flood requests static content such as videos. The POST Flood requires the server to lookup data in the databases.

Protocol AttackSYN Flood

One specific sub-attack of Protocol attacks is known as a SYN Flood. The SYN Flood attack takes advantage of an exploit in the TCP connection sequence. The request to initiate a TCP connection with the server must be confirmed by the requester. In the case of a SYN Flood, the confirmation is never given leaving the server waiting.

Volumetric AttackUDP Flood

A common method of Volumetric Attack is the UDP Flood. A UDP Flood floods a server with User Datagram Protocol Packets. The server checks to see if there are applications associated with the UDPs and responds with a “Destination Unreachable” packet.
